## Vision

### 10x Check
The 10x version is a content creation platform: AI generates carousel scripts from a topic prompt, templates are saved and shared via a team library (Supabase), and a mobile companion lets Brand Ambassadors create carousels on the go. The current plan's data layer supports template sharing and persistence without rewriting components; analytics and a mobile companion would require additional architecture beyond what's planned here.

## Accepted Scope (added to this plan)

Each item mapped to its build phase with acceptance criteria:

- **Paste-and-Go (Phase 4):** Live auto-parsing on paste/edit, debounced 300ms. Done when: typing in the textarea updates the slide preview without clicking a button.
- **Undo/Redo (Phase 1 types + Phase 5 wiring):** `HistoryState<T>` wrapper type defined in Phase 1. Cmd+Z / Cmd+Shift+Z wired in Phase 5. Done when: every CarouselState mutation (text edit, reorder, theme change, style change) is undoable/redoable.
- **Theme Preview Thumbnails (Phase 5):** Mini rendered hook slide at 1/10 scale per theme in the theme picker. Done when: the user can visually compare all themes at a glance without selecting each one.
- **Copy to Clipboard (Phase 3):** One-click copy via `ClipboardItem` with promise-based blob. Chrome-first; non-Chrome browsers fall back to PNG download. Done when: clicking "Copy" in Chrome puts the slide image on the clipboard.
- **Export Filename Templating (Phase 3):** Filenames use slugified carousel title + theme + date. Done when: exported files have meaningful names instead of generic `carousel.zip`.
